# 🪖 Smart Safety Helmet using IoT

An IoT-based Smart Safety Helmet designed to improve worker safety in industries such as construction, mining, and manufacturing. The helmet monitors environmental and motion-related parameters using multiple sensors and sends real-time alerts in case of hazardous conditions.

## 📌 Features

- Helmet wearing detection
- Fall/Accident detection
- Hazardous gas detection
- GPS location tracking
- Emergency SOS button
- Real-time IoT monitoring
- Buzzer and LED alerts

---

## 🛠 Hardware Components

- ESP32
- MPU6050
- MQ-2 / MQ-135 Gas Sensor
- DHT11 Sensor
- Neo-6M GPS Module
- Buzzer
- LEDs
- Push Button
- Battery Pack

## 🚀 How it Works

1. Sensors collect real-time data.
2. ESP32 processes sensor values.
3. Hazardous conditions trigger alerts.
4. GPS location is sent during emergencies.
5. Sensor data is uploaded to the IoT dashboard.
